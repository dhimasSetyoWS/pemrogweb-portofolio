Project Overview

Website Coca-Cola adalah proyek latihan (fake project) yang bertujuan untuk mengasah kemampuan dalam membangun antarmuka web yang responsif dan interaktif. Saya mengembangkan website ini menggunakan HTML, CSS, dan Bootstrap tanpa framework frontend modern, fokus pada desain yang menyerupai tampilan brand Coca-Cola yang ceria, merah, dan dinamis.

Proyek ini tidak terhubung ke backend atau database, namun menjadi portofolio penting yang menunjukkan kemampuan saya dalam layouting, grid system, komponen Bootstrap, dan efek interaktif sederhana (hover, carousel, modal).

Research & Design Reference

Meskipun bersifat latihan, saya tetap melakukan riset visual:

  • Mempelajari website resmi Coca-Cola (coca-cola.com) untuk menangkap palet warna, tipografi, dan gaya hero section.
  • Mengumpulkan inspirasi dari situs brand minuman lainnya seperti Pepsi, Fanta, dan Sprite.
  • Menentukan struktur halaman yang umum: navbar, hero, produk, tentang, footer.
  • Memastikan desain responsive dengan pendekatan mobile-first menggunakan Bootstrap 5.

Keputusan desain utama: menggunakan warna merah khas (#F40009), putih, dan hitam, dengan aksen curve yang merepresentasikan bentuk botol Coca-Cola.

Development & Implementation

Proyek ini dibangun dengan stack sederhana namun efektif:

  • HTML5 untuk struktur semantik.
  • CSS3 kustom untuk styling tambahan (animasi hover, efek bayangan, font styling).
  • Bootstrap 5 (grid system, navbar collapse, carousel, cards, modal).
  • Fitur interaktif: Carousel produk, tombol "Learn More" dengan modal, smooth scroll, dan form kontak sederhana.
  • Responsif: Diuji pada berbagai ukuran layar (mobile, tablet, desktop).

Kode dikelola di GitHub dan di-deploy menggunakan GitHub Pages sehingga dapat diakses publik.
Link live demo: https://dhimassetyows.github.io/cocaCola-page/

Key Outcomes & Learnings

Melalui proyek latihan ini, saya mendapatkan:

  • Pemahaman lebih dalam tentang Bootstrap grid dan utility classes untuk mempercepat development.
  • Pengalaman menciptakan desain responsif yang tetap rapi di segala perangkat.
  • Kemampuan mengintegrasikan komponen interaktif (carousel, modal, tooltip) tanpa JavaScript murni.
  • Portofolio visual yang dapat ditunjukkan ke calon klien atau rekruter.
  • Feedback positif dari teman-teman desainer bahwa tampilan terasa "seperti website Coca-Cola sungguhan".

Proyek ini menjadi fondasi untuk proyek-proyek frontend saya selanjutnya yang lebih kompleks (misal: dengan Vue.js atau React).
