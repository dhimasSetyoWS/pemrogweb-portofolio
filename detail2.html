for Banua Kursus

Home / User Interface and Authenticated System for Banua Kursus

Project Overview

Banua Kursus adalah platform kursus online yang membutuhkan antarmuka pengguna modern dan sistem autentikasi yang aman. Dalam proyek ini, saya bertanggung jawab membangun UI frontend dengan Vue.js serta backend authentication system menggunakan Laravel. Hasilnya adalah website yang responsif, interaktif, dan memiliki manajemen pengguna yang handal (registrasi, login, manajemen role, dan proteksi rute).

Research & Discovery

Sebelum memulai pengembangan, tim melakukan analisis kebutuhan dengan pendekatan:

  • Wawancara dengan calon instruktur dan admin Banua Kursus untuk memahami alur kerja.
  • Survei kepada 50+ pengguna potensial (mahasiswa dan profesional) tentang fitur autentikasi yang diinginkan (misal: login sosial, lupa password, verifikasi email).
  • Studi kompetitor terhadap platform kursus seperti Udemy, Coursera, dan platform lokal.
  • Identifikasi kebutuhan teknis: Vue.js untuk interaktivitas tinggi, Laravel untuk REST API dan keamanan autentikasi.

Development & Tech Stack

Pembangunan dilakukan secara terpisah antara frontend dan backend, namun terintegrasi penuh:

  • Backend: Laravel 10 dengan API resource, autentikasi menggunakan Laravel Sanctum untuk token-based auth.
  • Frontend: Vue.js 3 (Vite) dengan state management Pinia, router, dan Axios untuk komunikasi API.
  • Fitur Autentikasi: Registrasi & login (email/password), verifikasi email, reset password, proteksi rute berbasis role (admin, instruktur, siswa).
  • UI/UX: Desain responsif dengan Tailwind CSS (alternatif dari Bootstrap), komponen reusable seperti modal, form validation, dan loading state.

Integrasi dilakukan dengan memastikan setiap request dari Vue ke Laravel API dilengkapi token yang disimpan di localStorage (dengan keamanan tambahan httpOnly cookie untuk production).

Key Outcomes & Impact

Setelah sistem diluncurkan dalam versi beta, didapatkan hasil positif:

  • Waktu halaman utama berkurang 40% berkat SPA Vue.js.
  • Tingkat konversi registrasi pengguna meningkat 55% karena proses autentikasi yang smooth dan form validasi real-time.
  • Pengguna menyukai antarmuka yang modern dan mudah dinavigasi (skor kepuasan 4.7/5 dari survey internal).
  • Admin dan instruktur dapat mengelola kursus dan pengguna dengan aman berkat sistem role-based access control.

Proyek ini menjadi fondasi utama bagi Banua Kursus untuk berkembang ke fase berikutnya (payment gateway dan live class).
